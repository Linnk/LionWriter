Lion Writer project.

Lion Writer is a static content management for hipster hackers and/or anyone who hates Wordpress —or similar software— enough to learn this “pure” style of writting.

And I just realize that it works great for documentation sites!

There is other systems like Jekyll written on Ruby or Pelican written on Python. These are great static blog generators and those are great languages (very popular on the hacker community, unlike PHP), but a lot of us we actually have at least one or several servers running the old and good Apache/PHP.

The next steps are Dropbox integration and create a nice portrait page. If you notice that the design is inspired from Medium and you have taste for user interfaces, give me a hand to make it better. :)
